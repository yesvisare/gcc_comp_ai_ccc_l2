========================================

Business Context:
- Location: India GCC serving 50+ global business units
- Industry: IT services
- Data: Corporate documents, code repositories, technical documentation
- Required Frameworks: ISO 27001, SOC 2, GDPR (for EU clients)
- Multi-Tenant: 50+ business units across 15 countries
- Annual Revenue: ‚Çπ500 Cr

RAG Architecture:
- Components: Qdrant vector DB, Sentence Transformers, LLaMA 2, Kubernetes-based API
- Data Flows: Multi-tenant ingestion ‚Üí Tenant isolation ‚Üí Embedding ‚Üí Storage ‚Üí Retrieval
- Encryption: Per-tenant encryption keys with KMS
- Access Control: Tenant-aware RBAC, per-BU MFA policies

- SOC 2: 85% compliant (confidentiality TSC needs key rotation)

- 50+ distinct compliance profiles (each BU has different framework requirements)
- Chargeback model: ‚Çπ750-1,200/month per tenant
- 5-10 audits per year across frameworks
- Cross-tenant isolation verification critical

Estimated Remediation:
- Timeline: 12 weeks
- Cost: ‚Çπ18L
- Priority Gaps:
  1. Incident management procedures (28 hours)
  2. Key rotation automation (40 hours)

5. Multi-Framework Overlapping Controls Example
========================================

This example demonstrates how implementing ONE control can satisfy MULTIPLE frameworks:

Control: AES-256 Encryption at Rest

Satisfies:
- GDPR Article 32: Security of Processing
- SOC 2 Confidentiality TSC: C1.1
- ISO 27001 A.10: Cryptographic Controls
- HIPAA 164.312(a)(2)(iv): Encryption and Decryption

Implementation Effort: 24 hours (one-time)
Compliance Benefit: Satisfies 4 frameworks simultaneously
Cost Savings: Avoids 3 redundant implementations (72 hours saved)

Other Overlapping Controls:
1. Audit Logging (GDPR + SOC 2 + ISO + HIPAA)
2. RBAC Access Control (GDPR + SOC 2 + ISO + HIPAA)
3. MFA (SOC 2 + ISO + HIPAA)
4. Backup Strategy (SOC 2 + ISO + HIPAA)
5. Incident Response Plan (ISO + HIPAA)

Total Controls Across 4 Frameworks: 400
Unique Controls After Deduplication: 150
Reduction: 62.5%
